# maMeteo
Application Mobile réalisée en <b>ionic - Vue</b>.
<br>
<p>Permet d’afficher la météo de&nbsp;:</p>
<ul>
<li> Montréal</li>
<li>Laval</li>
<li>Québec</li>
<li>Et de la position où l’on se trouve (Géolocalisation)</li>
</ul>

Utilise le service <b>OpenWeatherMap</b> pour obtenir la météo.
